# 闲鱼黑胶卖家对比技能

## 功能
对比两个闲鱼卖家的黑胶唱片在售商品，找出重叠和独有的专辑。

## 文件说明

| 文件 | 说明 |
|------|------|
| [xianyu-vinyl-comparison.md](xianyu-vinyl-comparison.md) | 技能文档 |
| [xianyu_compare.py](xianyu_compare.py) | Python版本 |
| [xianyu_compare.js](xianyu_compare.js) | JavaScript版本（配合Playwright MCP） |

## 使用方法

### 方法1：直接运行（已登录状态）
确保浏览器已登录闲鱼，然后使用Playwright MCP执行抓取和对比。

### 方法2：新电脑/未登录
1. 打开闲鱼网页
2. 扫码登录
3. 输入"继续"后自动执行抓取

## 输出格式

```
========================================
   闲鱼卖家黑胶唱片对比
========================================

【统计】
  音乐大同: 178 张
  梦的采摘员: 167 张
  重叠: 59 张
  音乐大同独有: 116 张
  梦的采摘员独有: 108 张

【重叠专辑】(59张)
  1. 专辑名
  ...
========================================
```

## 默认卖家
- 卖家1: 音乐大同 (userId=2219735146783)
- 卖家2: 梦的采摘员 (userId=1059107164)

可修改CONFIG更换其他卖家。
